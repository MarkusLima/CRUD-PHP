## Installation

Rum this project

```bash
  create database cadastro;
```

```bash
  CREATE TABLE `cadastro`.`usuarios` (
  `id` INT NOT NULL AUTO_INCREMENT,
  `nome` VARCHAR(45) NULL,
  `email` VARCHAR(45) NULL,
  `senha` VARCHAR(45) NULL,
  PRIMARY KEY (`id`));
```
